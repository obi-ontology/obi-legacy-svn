OBI.owl Release Notes - this file relates to changes to the ontology file
owl:versionInfo 0.4, OBI SVN Revision 10.

These notes refer to the first of three steps taken to import of BFO into OBI.
Expanded notes on this process and rationale can be found on the wiki page:
https://www.cbil.upenn.edu/obiwiki/index.php/BFOImportStatus


Modifications in Step 1:
This file just reflects the name change from "FuGO" to "OBI".

Changes in FuGO.owl file header:
xmlns="http://fugo.sourceforge.net/ontology/FuGO.owl#"
 to
xmlns="http://obi.sourceforge.net/ontology/OBI.owl#"

Changed
  xml:base="http://fugo.sourceforge.net/ontology/FuGO.owl">
to
  xml:base="http://obi.sourceforge.net/ontology/OBI.owl">

Changed the ontology description to:
<dc:description rdf:datatype="http://www.w3.org/2001/XMLSchema#string"
    >The Ontology for Biomedical Investigation (OBI) is build in a collaborative, international effort and will serve as a resource for annotating biomedical investigations, including the study design, protocols and instrumentation used, the data generated and the types of analysis performed on the data. This ontology arose from the Functional Genomics Investigation Ontology (FuGO) and will contain both terms that are common to all biomedical investigations, including functional genomics investigations and those that are more domain specific.</dc:description>

Changed dc:title value to "Ontology for Biomedical Investigation"
Changed owl:VersionInfo value to "0.4"
Changed dc:date value to "2006-09-18"

Changes in rest of the owl file:
Substituded  string "FuGO_" with "OBI"

Changes in FuGO class names:
Changed names of 'organism' subclasses to reflect the names of the classes they were taken from (deleted the "_organism" postfix and made singular nouns)
